Question No 1

What is DOM and what is the purpose of DOM?

DOM stands for Document Object Model . It is an interface for web document to represent the page. It represents document as node and objects and can change the the document structure, style, and content.

Question No 2

How will you select HTML elements using DOM? Name the DOM methods.

We can select HTML elements by using selectors .These are getElementsByTagName(),getElementsByClassName(),getElementById(),querySelector(),querySelectorAll()

Question No 3

What is event bubble?

It is a DOM event propagation where an element receives an event and it bubbles up to its parent & ancestor elements in the DOM tree until it gets to the root element.

Question No 4

What is a callback function and why will you use it?

It is a function that passed into another function as an argument. We will use it if we want to execute one function after another function has completed its execution.
