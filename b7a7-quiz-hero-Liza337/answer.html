Q1. Differentiate among map , filter ,foreach and find

map() returns a new array while forEach() returns undefined
filter() returns a new array with those elements that satisfy the certain criteria while find() returns the first element that satisfy the criteria.

Q2. Differentiate among local storage ,session storage and cookies

local storage stores data with no expiration date while in session storage data is lost while close the tab and in cookies data are never expired until remove it manually.

Q3. What is eventLoop and its functionality

Event loop is something that allows javascript to handle thousands and thousands of requests with just a single thread. It is responsible for executing the code, collecting and processing events, and executing queued sub-tasks.
